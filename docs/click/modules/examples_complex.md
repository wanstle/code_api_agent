# examples/complex

### 顶层模块划分与职责
- **examples/complex**: 包含复杂命令行工具的示例，展示如何构建和使用复杂的命令行接口。

### 技术栈
- **Python**: 用于实现命令行工具。

### 关键组件之间的关系与数据流
- **ComplexCLI**: 顶层命令行接口，负责解析命令并调用相应的子命令。
- **Environment**: 环境管理类，提供环境相关的配置和状态。
- **cmd_init**: 初始化子命令，负责初始化环境。
- **cmd_status**: 状态子命令，负责显示当前环境的状态。

### 对外提供的能力
- 提供一个复杂的命令行接口，支持初始化和查看环境状态。
- 通过子命令实现模块化和可扩展性。
